## Responsive Behavior

The form is fully responsive using Tailwind CSS:
- Mobile: Full width, stacked layout
- Tablet: Optimized spacing
- Desktop: Max width constrained (`max-w-3xl`)

All new elements follow the same responsive patterns as existing form elements.

## Accessibility Features

1. **Proper Labels:**
   - All inputs have associated `<label>` elements
   - Required fields marked with red asterisk

2. **Error Messages:**
   - Displayed below input fields
   - Red color for visibility
   - Associated with input via `@error` directive

3. **Visual Indicators:**
   - Color is not the only indicator
   - Icons + text provide context
   - Clear messaging

4. **Keyboard Navigation:**
   - All form controls accessible via Tab
   - Proper focus states maintained
